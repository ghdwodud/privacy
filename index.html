개인정보처리방침

PDF Reader 앱은 사용자의 개인정보 보호를 최우선으로 합니다.

1. 개인정보 수집 및 이용

본 앱은 다음과 같은 정보를 수집할 수 있습니다:

  • 로컬 파일 접근: 사용자가 선택한 PDF 파일에 대한 접근 권한
  • 앱 사용 통계: 앱의 성능 개선을 위한 익명화된 사용 통계
  • 광고 식별자: Google AdMob을 통한 맞춤형 광고 제공 (선택적)

2. 개인정보 수집 목적

  • PDF 파일 읽기 및 표시 기능 제공
  • 앱 성능 개선 및 오류 수정
  • 사용자 경험 향상을 위한 기능 개발
  • 광고 서비스 제공 (Google AdMob 사용 시)

3. 개인정보 보관 및 처리

  • 로컬 저장: 모든 PDF 파일은 사용자의 기기에만 저장됩니다
  • 외부 전송 없음: 사용자의 PDF 파일은 외부 서버로 전송되지 않습니다
  • 임시 데이터: 앱 사용 중 생성되는 임시 데이터는 앱 종료 시 자동 삭제됩니다

4. 제3자 서비스

본 앱은 다음 제3자 서비스를 사용할 수 있습니다:

  • Google AdMob: 광고 서비스 제공 (Google의 개인정보처리방침 적용)
  • Google Analytics: 앱 사용 통계 수집 (익명화된 데이터만)

5. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 개인정보 수집 및 이용에 대한 동의 철회
  • 개인정보 열람, 정정, 삭제 요청
  • 광고 개인화 설정 변경 (기기 설정에서 가능)

6. 보안 조치

  • 로컬 파일 암호화 저장
  • 안전한 데이터 전송 프로토콜 사용
  • 정기적인 보안 업데이트

7. 개인정보처리방침 변경

본 개인정보처리방침은 필요에 따라 변경될 수 있으며, 중요한 변경사항이 있을 경우 앱 내 공지 또는 이메일을 통해 안내드립니다.

8. 문의사항

개인정보처리방침에 대한 문의사항이 있으시면 다음으로 연락해 주세요:

  • 이메일: [여기에 이메일 주소 입력]
  • 앱 내 문의: 설정 > 문의하기
